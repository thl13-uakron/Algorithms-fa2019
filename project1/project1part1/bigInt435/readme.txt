README

	This is program that generates random RSA-435 encryption keys. 


	To execute this program:

		1) Generate the executable by entering "make all" into the terminal
		while inside this directory

		2) Open the "rsa435" executable to run the code


	Upon execution:

		1) Two large, randomly generated prime numbers will be recorded into 
		a file named "p_q.txt"

		2) The public key pair calculated based on the above will be recorded
		into a file named "e_n.txt"
		
		3) The private key pair calculated based on both of the above will be 
		recorded into a file name "d_n.txt"
